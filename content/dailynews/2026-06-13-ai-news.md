### OpenAI 革新 Codex 速率限制机制：可储存、可赠送、可手动触发

OpenAI 于 6 月 12 日为其 AI 编程智能体 Codex 推出全新的速率限制重置规则，彻底改变了此前「到点自动重置」的固定模式。新机制允许用户将重置次数存入「重置额度账户」，在需要时手动触发，大幅提升了使用灵活性。

具体规则如下：
- **全套餐覆盖**：Go、Plus、Pro、商业版用户均可获得 1 次免费的速率限制重置资格
- **邀请扩容**：Plus/Pro 用户在未来两周内邀请最多 3 名好友试用 Codex，双方各得 1 次额外重置额度
- **手动触发**：储存的重置额度可随时使用，避免业务高峰期因速率限制被中断

此次更新被视为 OpenAI 与 Anthropic 之间「AI 价格战」的开端。此前 AI 使用成本大幅上涨，OpenAI CEO Sam Altman 近期也公开表示「AI 成本已成为企业的重大问题」，降低成本、提升灵活性成为争夺开发者流量的核心策略。

**核心亮点：**
- 速率限制从「被动等待」升级为「主动规划」
- 邀请机制带来病毒式传播潜力，直接对标 Claude Code 的用户粘性
- 标志着 AI 编程工具从「能力竞赛」转向「开发者体验竞赛」

### OpenAI 祭出免费 Codex 迁移：两月免费 + 一键导入，直击 Claude Code 软肋

OpenAI 于 5 月 13 日推出「Switch to Codex」促销活动，为从竞品（主要是 Anthropic Claude Code）迁移的企业用户提供 **2 个月 Codex 企业版免费使用权**。申请窗口约在 6 月中旬截止，正值当前时间点。

更值得关注的是 OpenAI 同步推出的一键迁移工具：用户在 Codex 设置中点击「导入其他代理配置」，工具自动扫描设备上的 Claude Code 配置，可迁移内容包括：
- 自定义技能、钩子（Hooks）
- MCP（模型上下文协议）服务器配置
- 子代理设置、指令文件
- 最多 30 天的会话历史记录

此前 Claude Code 的核心壁垒是用户积累的自定义配置和切换成本，OpenAI 的迁移工具直接将这部分「锁定效应」消解。社区也出现了开源工具 `claude2codex`，支持单条命令完成配置转换。

**核心亮点：**
- 免费 2 个月 + 一键迁移，双管齐下拆解 Claude Code 用户粘性
- 直接攻击 Anthropic 增长最快的营收来源（Claude Code 订阅）
- AI 编程工具「Agent 配置标准化」趋势初现，MCP 生态成关键战场

### 国内大厂 AI 高考志愿 Agent 全面开战：阿里千问领跑，腾讯百度字节跟进

2026 年高考志愿填报季，阿里、腾讯、百度、字节四大厂集体入局，纷纷推出免费 AI 志愿填报智能体，将这场垂直场景之战推向高潮。

**阿里千问**于 6 月 10 日上线国内首个全周期高考志愿填报 Agent，具备「志愿报告」「志愿日历」「志愿问答」三项核心能力，基于千问高考志愿大模型打造。2025 年该大模型已累计为考生免费提供 1300 万份志愿报告。

**腾讯「元宝高考通」**加码 Agent 式深度搜索和多轮对话能力；**百度**依托搜索数据优势走权威性路线；**字节**则借助豆包的内容生态覆盖考生决策全链路。

艾媒咨询预计，2026 年中国高考志愿填报市场付费规模将达 **11.6 亿元**。大厂的免费模式正在重塑行业格局：传统机构靠信息差盈利的低端测算需求被 AI 承接，线下机构转向更深层的生涯规划，「AI 辅选、专家兜底」的新商业模式正在形成。

**核心亮点：**
- 高考志愿成为大厂 AI Agent 落地的重要垂直场景
- 从「大模型问答」升级为「智能体主动规划」，产品形态迭代明显
- 免费模式抢夺家庭刚需流量，AI 下沉至县域及农村考生群体
