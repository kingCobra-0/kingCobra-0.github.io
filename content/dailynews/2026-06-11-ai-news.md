### OpenAI 更新 Responses API，Agent 任务原生支持上线

OpenAI 对 Responses API 进行重大更新，**新增 Agent 任务原生支持能力**，允许模型自主规划、调用工具并完成多步骤任务。同时优化高风险代码生成请求的分发逻辑，由更稳定的 GPT-5.2 模型接管高风险 GPT-5.3-Codex 请求。

**核心亮点：**
- Responses API 新增原生 Agent 任务支持，模型可自主规划并完成多步骤任务
- 高风险 GPT-5.3-Codex 请求自动路由至 GPT-5.2 执行，平衡能力与安全性
- 为 Agentic 应用开发者提供更稳定的底层 API 能力
